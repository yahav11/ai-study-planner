
# AI Study Planner

A Streamlit web app that generates a personalized study plan using OpenAI.

## How to use
1. Enter a subject
2. Choose your level
3. Pick number of days
4. Click "Generate study plan"

## Tech
- Python
- Streamlit
- OpenAI API
